# ShikshaAI Bharat - Frontend

This is the frontend repository for **ShikshaAI Bharat**, a multilingual, voice-first educational platform tailored for Indian students. It provides an intuitive, modern, and highly interactive user interface designed to feel like an intelligent classroom copilot.

This frontend is completely decoupled from the AI backend and is designed to be hosted globally (e.g., via Vercel), while dynamically communicating with a local or cloud-hosted AI inference server via secure tunneling.

## Tech Stack

* **Framework:** [Next.js](https://nextjs.org/) (App Router)
* **Language:** TypeScript
* **Styling:** Tailwind CSS
* **Icons:** Lucide React
* **Hosting:** Vercel (Recommended)
* **Audio Capture:** Custom browser-based `MediaRecorder` API integrations.

## Core Features

1. **Voice-First Interaction:** Students can tap the microphone to speak naturally in Hinglish, Tamil, Hindi, English, and more. 
2. **AI Simplify:** Converts complex technical concepts into grade-level appropriate explanations.
3. **Visual Learning:** Dynamically renders AI-generated educational diagrams and flowcharts directly in the browser.
4. **Quiz Studio:** Parses and renders JSON-based MCQ and short-answer assessments generated by the backend.
5. **Session History:** Automatically persists past explanations, diagrams, and quizzes locally using `localStorage` for fast retrieval via the Sidebar.

## Installation & Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/Jack-ksrct/shikshaai-frontend.git
   cd shikshaai-frontend
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Configure Environment Variables:**
   Create a `.env.local` file in the root of the project and set your backend API URL. If your backend is running locally via Ngrok, paste the Ngrok URL here.
   ```env
   NEXT_PUBLIC_API_URL=https://your-backend-ngrok-url.ngrok-free.app
   ```

4. **Start the development server:**
   ```bash
   npm run dev
   ```
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.

## Deployment

This project is optimized for deployment on **Vercel**. 

1. Connect your GitHub repository to Vercel.
2. Add the `NEXT_PUBLIC_API_URL` variable in your Vercel Project Settings -> Environment Variables.
3. Deploy!

*Note: Since the frontend relies on a dynamically generated Ngrok URL for the backend, you must update the `NEXT_PUBLIC_API_URL` environment variable in Vercel and trigger a redeployment whenever your backend Ngrok URL changes.*
